# Place your dashboard screenshots here

After building the dashboard in Power BI Desktop, export screenshots of each page and add them to this folder.

**Suggested naming convention:**
- `01_executive_summary.png`
- `02_sales_by_region.png`
- `03_category_analysis.png`
- `04_trends_over_time.png`
- `05_customer_segments.png`
- `06_top_products.png`

Screenshots will be referenced in the main README.md once they are available.
