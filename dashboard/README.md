# Place your Power BI Desktop file here

Add the `.pbix` file produced in Power BI Desktop to this folder.

**Suggested filename:** `PowerBI_Dashboard.pbix`

## How to open

1. Download and install [Power BI Desktop](https://powerbi.microsoft.com/desktop/) (free).
2. Open the `.pbix` file in Power BI Desktop.
3. If the data source path has changed, update it in **Home → Transform Data → Data Source Settings**.
4. Click **Refresh** to reload data from `data/sales_data.csv`.
