# Power BI Dashboard Project
This project presents an interactive Power BI dashboard created to analyze key business data and turn it into clear, actionable insights.
<img width="2617" height="1464" alt="intro" src="https://github.com/user-attachments/assets/dd915a75-9476-4363-a623-3851aeef4aae" />


## Project Objective
The goal of this project was to explore the dataset, identify important trends and performance patterns 
and present the results through a clear and user-friendly dashboard.

## Tools Used
- Power BI
- Excel / CSV
- Data cleaning
- Data visualization
- Dashboard design


## Dashboard Features
- KPI summary cards
- Trend analysis
- Category or segment breakdown
- Regional / product / customer insights
- Interactive filters and slicers


## What I Learned
This project helped me strengthen my skills in data visualization, dashboard storytelling, KPI design, and presenting insights in a clear and structured way.
